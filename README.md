# Cldr Calendars

> My wife's jealousy is getting ridiculous. The other day she looked at my calendar and wanted to know who May was.
> -- Rodney Dangerfield

## Installation

Add `ex_cldr_calendars` to your `deps` in `mix.exs`:

```elixir
def deps do
  [
    {:ex_cldr_calendars, "~> 0.1.0"}
    ...
  ]
end
```
